# Sources

- Primary source URL: https://nces.ed.gov/pubs2000/2000488.pdf
- Source family: NAEP / NCES (official release)
- Reuse note: NCES publication. Unless specifically noted, NCES information is in the public domain.
- Prompt/rubric recovery: prompt and six-level rubric recovered directly from the official NCES release.
- Dataset build note: student texts were transcribed from the released sample-response pages and normalized into UTF-8 plain text without changing wording or spelling. Line-wrap artifacts from the PDF were removed.
- Level mapping: Official NAEP ordinal scale mapped directly as Unsatisfactory=1, Insufficient=2, Uneven=3, Sufficient=4, Skillful=5, Excellent=6. Submissions are ordered strongest-to-weakest so gold_rank=1 is the strongest paper.
- Ranking rule: files are ordered strongest to weakest (s001 strongest, final file weakest).
