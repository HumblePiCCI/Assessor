
# End of Key Stage 1 English writing portfolios

- Source family: Standards and Testing Agency (STA), GOV.UK
- Assessment year: 2018 exemplification set
- Grade band: Key Stage 1 / Year 2
- Assessment unit: portfolio of independent writing pieces
- Cohort coherence: same grade band, same STA teacher-assessment framework, same portfolio collection form, explicit WTS / EXS / GDS judgements

## Portfolio form

Each submission is a teacher-assessment exemplar portfolio rather than a single prompt response. The benchmark unit is the full curated portfolio as released by STA. Portfolio pieces include narrative, description, explanation, letter/instructions, information, recount, procedural, book review, and character description depending on the pupil.
