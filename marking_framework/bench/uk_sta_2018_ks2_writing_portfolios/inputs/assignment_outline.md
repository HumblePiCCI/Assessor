
# End of Key Stage 2 English writing portfolios

- Source family: Standards and Testing Agency (STA), GOV.UK
- Assessment year: 2018 exemplification set
- Grade band: Key Stage 2 / Year 6
- Assessment unit: portfolio of independent writing pieces
- Cohort coherence: same grade band, same STA teacher-assessment framework, same portfolio collection form, explicit WTS / EXS / GDS judgements

## Portfolio form

Each submission is a full writing portfolio rather than a same-prompt essay. This is still benchmark-usable because the cohort shares grade band, framework family, and collection form. Across the four portfolios, forms include non-fiction, letter, prediction, description, balanced argument, narrative, biography, procedural writing, recount, information writing, diary, explanation, and newspaper report.
